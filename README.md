# OIBSIP Task 4 - Email Spam Detection with Machine Learning

## 📌 Project Overview
This project aims to classify emails as Spam or Ham (Not Spam) using Machine Learning techniques. The model is trained on a dataset of email messages and predicts whether a message is spam or legitimate.

## 🏢 Internship Details
- Organization: Oasis Infobyte
- Internship Program: OIBSIP
- Task Number: Task 4
- Project Name: Email Spam Detection with Machine Learning

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## 📂 Dataset
The dataset contains email messages labeled as:
- Spam
- Ham (Not Spam)

## 🔄 Project Workflow
1. Data Collection
2. Data Cleaning
3. Handling Missing Values
4. Label Encoding
5. Text Vectorization using TF-IDF
6. Train-Test Split
7. Model Training using Naive Bayes
8. Model Evaluation
9. Spam Prediction

## 📊 Model Used
- Multinomial Naive Bayes

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## ✨ Features
- Detects spam emails automatically
- Classifies emails into Spam and Ham categories
- Uses Machine Learning for prediction
- Provides performance evaluation metrics

## 🎯 Results
The model achieved high accuracy in detecting spam emails and effectively classified messages into spam and non-spam categories.

## 📝 Conclusion
This project demonstrates the use of Machine Learning and Natural Language Processing techniques for email classification. By applying TF-IDF Vectorization and Naive Bayes classification, the model successfully identifies spam emails with good accuracy.
